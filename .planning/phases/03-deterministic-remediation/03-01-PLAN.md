<task type="auto" tdd="true">
  <name>Task 1: Implement HeadingNormalizationSkill</name>
  <files>src/pdf_accessibility/skills/remediation/headings.py, tests/test_remediation_structural.py</files>
  <behavior>
    - If document starts with H3, first H3 becomes H1.
    - If H1 is followed by H3, H3 becomes H2 (no skipping levels).
    - If H1 is followed by H2, it remains H2.
    - If H2 is followed by H1, it remains H1 (jumping up is allowed).
  </behavior>
  <action>
    Create `HeadingNormalizationSkill` class inheriting from `RemediationSkill`.
    Implement `remediate` method:
    - Iterate through all pages and blocks in reading order.
    - Maintain `current_max_depth`.
    - Detect `heading1` to `heading6` roles.
    - If a heading skips a level down, adjust its role.
    - Record changes in `RemediationAction`.
  </action>
  <verify>
    <automated>pytest tests/test_remediation_structural.py -k test_heading_normalization</automated>
  </verify>
  <done>Heading hierarchy is normalized correctly across multiple pages.</done>
</task>
